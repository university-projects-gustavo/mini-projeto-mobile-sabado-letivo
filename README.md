# Repositório de atividade de aula DDM

Este repositório tem como objetivo **armazenar e organizar a atividade referente ao sábado letivo**.  

As atividades realizadas no vídeo abragem o passo a passo disponibilizado pelo professor:

- Com com o ambiente instalado foi executado o comando "flutter doctor" para verificar o ambiente;
- Foi criado um projeto flutter com o comando "flutter create teste_app" para verificar a estrutura;
- Foi dado uma breve explicação da pasta 'lib', arquivo 'main.dart', arquivo 'pubspec.yaml';
- Com o comando "flutter run -d chrome" foi simulado o android no chrome;
- Com o comando "flutter devices" para listar os dispositivos;
- Com o comando "flutter run -d [id do dispositivo móvel]" foi sumaldo o app em um verdadeiro dispositivo mobile;
